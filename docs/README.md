# Документация / Documentation

## Схема подключения / Wiring Diagram

Поместите изображение со схемой подключения в эту папку с именем:

**Place the wiring diagram image in this folder with the name:**

- `wiring-diagram.png` (рекомендуется / recommended)
- или / or `wiring-diagram.jpg`

Изображение будет автоматически отображаться в основном README.md файле.

**The image will automatically appear in the main README.md file.**
